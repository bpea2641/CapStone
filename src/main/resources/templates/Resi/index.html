어떤걸 즐길 수 있을까요?

AI를 이용한 질문, 자신만의 스토리, 원하는 텍스트를 입력하고 자신만의 사진을 AI가 당신에게 그려줄 수 있어요!

저희 사이트는 당신에게 AI의 무궁무진한 가능성을 보여줄 수 있습니다. 당신이 평소에 궁금했던 점이나, 혹은 어려운 문제에 쳐했을 때. CHAT AI가 당신을 도와 줄 수 있습니다. 혹은 자신이 원하는 그림이나 이미 가지고 있는 그림을 변형 시킬 수 있고, 사진속의 의미가 뭔지 AI가 알려줄 수 있습니다. 텍스트를 음성으로 바꾸거나, 정리를 위해 음성파일을 텍스트로 바꾸는 것 또한 가능합니다.

  • 여러 종류의 텍스트 문답 기능. 여러분이 원하는 질문을 마음껏 해보세요!
  • AI가 직접 제공하는 여러 테마의 사진이 기다리고 있습니다!
  • 평소 음성파일을 정리할 때 시간이 오래 걸리시나요? AI가 단숨에 정리해줍니다!
